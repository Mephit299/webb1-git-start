Arbetsflöde/gång

  1. Skapade en fork av ett repository
  2. Klonade repot med github desltop (gröna-knappen)
  3. Öppnade repot med vsc
  4. Öppnade filen index.html
  5. Ändrade i index.html, skapade listor och bild
  6. för att vissa bil gjorde vi en mapp och sparede bilden i den
  7. sen använde vi img för att visa bilden
  8. Vi commitade allting till github med en fin kommentar
  9. vi slog på github pages för att få en gratis webhost med allt
  10. vi laddade in en google font
